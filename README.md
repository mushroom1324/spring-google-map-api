# Meet-In-The-Middle
- 보안 내용은 아니고요.. 그냥 말장난입니다.

## Introduction

- 친구와 어디서 만날지 고민할 때 도움이 될 수 있는 Google Maps API 기반 중간점 서칭 API입니다.

## Features

- 내 위치 확인 기능
- 중간점 후보지 서치 기능
- '최적의 중간지' 선정 기능

## Why

- '공정함'이 당신의 만남에 가장 중요한 키워드라면 써보십시오.
- 유저 간 중간점을 찾고, 중간점들 중 양쪽에서 찾아가는 거리가 가장 비슷한 위치를 반환합니다.
